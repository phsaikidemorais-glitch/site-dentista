# 🦷 Sorriso Perfeito - Site Clínica Odontológica

Site institucional desenvolvido para clínica odontológica, com design moderno, responsivo e foco em conversão de pacientes.

## 🌐 Demo ao vivo

👉 [dentistapedro.vercel.app](https://dentistapedro.vercel.app)

## ✨ Funcionalidades

- Design moderno e responsivo (funciona em celular, tablet e desktop)
- Seção de serviços odontológicos
- Formulário de agendamento de consulta
- Botão flutuante de WhatsApp
- Informações de contato e horários
- Animações suaves e navegação fluida

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 com Tailwind CSS
- JavaScript
- Hospedagem na Vercel

## 📋 Sobre o projeto

Este é um projeto de portfólio que demonstra a criação de um site institucional completo para o segmento odontológico. Pode ser facilmente adaptado para qualquer clínica.
